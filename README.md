## Quick Start

### Option 1: With Cloudflare Tunnel (Recommended)

Run the start script to launch both the dev server and cloudflare tunnel:

```bash
./start-tunnel.sh
```

This will:
- Start the Vite dev server on port 5173
- Create a Cloudflare tunnel
- Display both local and public tunnel URLs
- Show the tunnel URL in the app interface

### Option 2: Local Development Only

```bash
npm run dev
```

Then open http://localhost:5173
